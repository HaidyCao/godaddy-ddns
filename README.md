# GODADDY-DDNS

更新godaddy的dns记录

## Params

    -N string
        new ip
    -T int
        domain name (default 600)
    -d string
        godaddy developer domain
    -k string
        godaddy developer key
    -n string
        domain name (default "@")
    -s string
        godaddy developer secret
    -t string
        domain type (default "A")
    -u string
        default check url (default "http://api.ipify.org")
